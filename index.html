ZoomCharts echo service

This service is used by ZoomCharts library to enable file download in browsers that do not support <a download> attribute.

This service does not store any data except for usage information needed to prevent service abuse. No chart image or data is ever stored.

It is recommended that you install this service on your own server to avoid the chart data from being sent to outside resources. Please see the GitHub repository for sources and instructions on how to do this.
